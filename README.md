# Unbeatable Tic Tac Toe

A Tic Tac Toe AI that uses a rule-based forced strategy to ensure it never loses.  
The bot prioritizes winning moves, blocks opponent threats, and follows optimal placement rules to guarantee a win or draw.

## Features
- Unbeatable AI (never loses)
- Rule-based decision making
- Optimal move prioritization

## How it works
The AI follows a set of predefined strategies such as winning when possible, blocking opponent moves, and choosing optimal positions (center, corners, edges) to avoid losing scenarios.
